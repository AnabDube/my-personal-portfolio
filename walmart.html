Walmart Sales Dashboard

Project Overview

Built an end-to-end sales analytics dashboard in Excel, integrating Power Query to automate cleaning and transformation of store-level transactions...

Challenge

Raw Walmart sales data was inconsistent and required automation to avoid repetitive manual cleaning each month...

Solution

Automated the entire data pipeline using Power Query and designed a dashboard with KPIs such as total sales, average monthly revenue, top stores, and seasonal patterns...

Impact

Reduced manual cleaning by 60%, provided leadership with near real-time sales insights, and enabled better decision-making on seasonal stock planning...
